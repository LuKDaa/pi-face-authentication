Proyecto: Control de Acceso Facial

Explicación del Proyecto

Este es un prototipo funcional de un sistema de control de acceso biométrico. Utiliza una Raspberry Pi 3B+ y una cámara para identificar rostros autorizados en tiempo real y, en caso de éxito, activar un relé o servomotor para desbloquear una puerta.

El proyecto está diseñado como un sistema integrado. El reconocimiento facial se realiza con OpenCV, utilizando el algoritmo LBPH (Local Binary Patterns Histograms), que es ideal por su ligereza y eficiencia en hardware limitado. El sistema incluye scripts dedicados para la captura de rostros, el entrenamiento del modelo y la ejecución del reconocimiento.

Para facilitar su administración, el sistema cuenta con un Panel de Administración Web desarrollado en Flask. Desde este panel, un administrador puede añadir nuevos usuarios, ver los registros de acceso (almacenados en una base de datos SQLite) y disparar el re-entrenamiento del modelo.

Tecnologías Utilizadas:

  • Python
  • OpenCV
  • Flask
  • SQLite
  • Raspberry Pi
  • GPIO
  • HTML/CSS
